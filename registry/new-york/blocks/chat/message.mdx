# Chat · Message

A single chat bubble, for a `user` or `assistant` turn. Part of the `Chat` block family (`Message`, `Composer`, `TypingIndicator`).

<Canvas of={MessageStories.Assistant} />

## When to use

- Rendering a conversation turn in an AI/chat surface.

## Roles

- `user` — the person's message.
- `assistant` — the AI's reply.

Optional `name` and `timestamp` add attribution. Keep alignment and styling consistent per role across the product.

<Canvas of={MessageStories.User} />

## Accessibility

Role is conveyed by more than position/colour (name + structure), so the conversation is followable by screen readers. Timestamps use a readable, absolute or clearly-relative format.
