# Chat · Composer

The prompt input for a chat surface. **Enter** sends, **Shift+Enter** adds a newline, and the Send button becomes **Stop** while the assistant is generating.

<Canvas of={ComposerStories.Default} />

## When to use

- The single input where the user types prompts to an assistant.

## States

- **Default** — ready for input.
- **Generating** — Send becomes Stop so the user can interrupt.
- **Disabled** — input blocked (e.g. quota reached); explain why in the placeholder.

<Canvas of={ComposerStories.Generating} />
<Canvas of={ComposerStories.Disabled} />

## Accessibility

The input is keyboard-first: Enter/Shift+Enter are standard and discoverable. The Send/Stop control is a real button with a clear accessible label that updates with state. Disabled state is conveyed to assistive tech, not just visually.
