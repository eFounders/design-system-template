# Stat

One key number, a label, and an optional trend. For dashboards and summary headers.

<Canvas of={StatStories.Default} />

## When to use

- Surfacing a single headline metric (revenue, active users, conversion).
- A row of comparable KPIs at the top of a dashboard.

## When not to use

- Long lists of numbers — use a table.
- Numbers that need context to read — pair with a chart instead.

## Trend

`trend` colours the delta: `up` → success, `down` → destructive, `neutral` → muted. The trend is a semantic signal, so it also reads without colour (the delta text and direction carry the meaning).

<Canvas of={StatStories.Row} />

## Accessibility

The label is always present and associated with its value, so the metric is meaningful to screen readers. Trend colour is reinforced by the delta sign, never colour alone.
